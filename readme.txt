	Bem-vindos(as) ao repositório do Grupo de Estudo para a Maratona de Programação, o GEMP. Esse grupo foi formado para auxiliar os alunos da Universidade Federal do Ceará, e os membros da Comunidade de Programadores, com uma atividade preparatória para competições de Algoritmos e Programação de Computadores, com foco na Maratona de Programação (SBC).	A Maratona de Programação é uma competição anual que ocorre no Brasil, sendo uma seletiva reginal para o ACM - International Collegiate Programming Contest, evento internacional que desafia equipes de programação de diversas regiões do planeta.

	Nesse repositório você encontrará Algoritmos, Códigos e Problemas que são recorrentes em concursos de Programação, e que desafiam as habilidades dos programadores, sempre através de enunciados criativos e raciocínios precisos.
