
	Algoritmo: Insertion_Sort
	Entrada: um vetor A[1..n]
	Pré-condição: N >= 0
	Saída: nenhuma
	Pós-condição: A[1..n] está ordenado.
	------------------------------------------------
	1. | PARA i DE 2 A n-1
	2. |  | chave := A[i]
	3. |  | j := i-1
	3. |  | ENQUANTO j > 0 E chave < A[j]
	4. |  |  | A[j+1] := A[j]
	5. |  |  | --j
	6. |  | A[j] := chave


Legenda:

  "|"  ->  Escopo de Hidentação 
  ":="  ->  Atribuição
